# primetradeai-trade-analysis
Exploratory Data Analysis (EDA) of Hyperliquid trading data to identify how Bitcoin market sentiment influences trading behavior, profitability, and risk.



# Project Overview

This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and Hyperliquid trader performance. The goal is to understand how market sentiment affects trading activity, profitability, and trader behavior.

# Dataset

This project uses two datasets:

1.Hyperliquid Historical Trading Data
2.Bitcoin Fear & Greed Index
Place both datasets inside the `data/` folder before running the notebook.

# Key Insights

- Fear had the highest number of trades.
- Greed showed the highest average profit per trade.
- Fear generated the highest total profit due to higher trading activity.
- Buy trades performed better during Fear.
- Sell trades performed better during Greed.
